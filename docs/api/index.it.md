# API

L'API di AirGap AI Chatbot fornisce accesso programmatico a conversazioni,
documenti e gestione dei workspace. Utilizzarla per integrare le capacita di chat
nelle proprie applicazioni, automatizzare flussi di lavoro o creare front-end
personalizzati. Questa sezione copre gli endpoint disponibili e come autenticare
le richieste.

Tutti gli endpoint API accettano e restituiscono JSON. L'API segue le convenzioni
REST per gli URL delle risorse e i metodi HTTP, con codici di stato standard per
le risposte di successo e di errore. Iniziare con il riferimento agli Endpoint
per vedere cosa e disponibile, quindi consultare Autenticazione per imparare a
proteggere le chiamate API.

- **[Endpoint](endpoints.md)** -- Endpoint API disponibili e formati di
  richiesta/risposta
- **[Autenticazione](authentication.md)** -- Gestione delle chiavi API e
  gestione dei token
