# API

The AirGap AI Chatbot API provides programmatic access to conversations,
documents, and workspace management. Use it to integrate chat capabilities into
your own applications, automate workflows, or build custom frontends. This
section covers the available endpoints and how to authenticate requests.

All API endpoints accept and return JSON. The API follows REST conventions for
resource URLs and HTTP methods, with standard status codes for success and error
responses. Start with the Endpoints reference to see what is available, then
review Authentication to learn how to secure your API calls.

- **[Endpoints](endpoints.md)** -- Available API endpoints and request/response
  formats
- **[Authentication](authentication.md)** -- API key management and token
  handling
