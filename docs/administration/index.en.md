# Administration

Manage users, control access, and configure system-wide settings for AirGap AI
Chatbot. This section is intended for administrators responsible for deploying,
configuring, and maintaining the platform. It covers role-based access control
and global application settings.

The RBAC system lets you define who can access what resources and what actions
they can perform. The Settings page provides control over system configuration
including API keys, default model parameters, and integration options.

- **[RBAC](rbac.md)** -- Configure role-based access control and user
  permissions
- **[Settings](settings.md)** -- Manage system configuration and API keys
