# Amministrazione

Gestire utenti, controllare gli accessi e configurare le impostazioni di sistema
per AirGap AI Chatbot. Questa sezione e destinata agli amministratori responsabili
della distribuzione, configurazione e manutenzione della piattaforma. Copre il
controllo degli accessi basato sui ruoli e le impostazioni globali
dell'applicazione.

Il sistema RBAC permette di definire chi puo accedere a quali risorse e quali
azioni puo eseguire. La pagina Impostazioni fornisce il controllo sulla
configurazione di sistema, incluse le chiavi API, i parametri predefiniti del
modello e le opzioni di integrazione.

- **[RBAC](rbac.md)** -- Configurare il controllo degli accessi basato sui
  ruoli e le autorizzazioni utente
- **[Impostazioni](settings.md)** -- Gestire la configurazione di sistema e le
  chiavi API
