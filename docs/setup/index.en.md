# Setup

Get AirGap AI Chatbot running on your infrastructure. This section walks you
through pulling the container image, running the application, and configuring it
to match your environment requirements. Whether you are deploying on a single
machine or preparing a production installation, these guides cover the essential
first steps.

After completing the setup process, you will have a running instance ready for
your first conversation. The Installation guide covers the Docker pull and run
commands, while the Configuration guide explains environment variables, volume
mounts, and application settings.

- **[Installation](installation.md)** -- Pull and run the AirGap AI Chatbot
  container
- **[Configuration](configuration.md)** -- Customize settings for your
  environment
