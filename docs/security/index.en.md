# Security

Understand the security architecture, threat model, and compliance features of
AirGap AI Chatbot. This section is essential for administrators and security teams
evaluating the platform for regulated or sensitive environments. It covers how
data is handled, what threats are mitigated, and what compliance standards are
supported.

Security in an air-gapped context means more than just network isolation. AirGap AI
Chatbot is designed to protect data at rest and in transit, enforce access controls,
and maintain audit trails. The Overview page details the threat model and data
handling practices, while Compliance covers GDPR, data residency, and audit
logging capabilities.

- **[Overview](overview.md)** -- Threat model, data handling, and security
  architecture
- **[Compliance](compliance.md)** -- GDPR, data residency, and audit logging
