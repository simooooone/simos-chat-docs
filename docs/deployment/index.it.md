# Distribuzione

Scegliere il metodo di distribuzione adatto al proprio ambiente. Questa sezione
copre sia le distribuzioni standard basate su Docker sia le installazioni
completamente air-gapped per ambienti con restrizioni. Che si stia distribuendo
su un server cloud, un data center on-premises o una rete scollegata, queste
guide illustrano il processo passo dopo passo.

La guida Docker e il punto di partenza consigliato per la maggior parte degli
utenti. Utilizza Docker Compose per configurare l'intero stack con un singolo
comando. Se il proprio ambiente non ha accesso a Internet, la guida alla
Configurazione Air-Gap spiega come trasferire e installare tutto offline.

- **[Docker](docker.md)** -- Distribuire con Docker e Docker Compose
- **[Configurazione Air-Gap](air-gap.md)** -- Configurare una distribuzione
  offline per ambienti con restrizioni
