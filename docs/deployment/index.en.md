# Deployment

Choose the deployment method that fits your environment. This section covers
both standard Docker-based deployments and fully air-gapped installations for
restricted environments. Whether you are deploying to a cloud server, an
on-premises data center, or a disconnected network, these guides walk you through
the process step by step.

The Docker guide is the recommended starting point for most users. It uses
Docker Compose to set up the full stack with a single command. If your
environment has no internet access, the Air-Gap Setup guide explains how to
transfer and install everything offline.

- **[Docker](docker.md)** -- Deploy with Docker and Docker Compose
- **[Air-Gap Setup](air-gap.md)** -- Set up an offline deployment for
  restricted environments
