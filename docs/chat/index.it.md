# Chat

L'interfaccia di chat e dove si interagisce con AirGap AI Chatbot quotidianamente.
Questa sezione copre tutto, dall'invio del primo messaggio all'utilizzo di
funzionalita avanzate come allegati e gestione del contesto. Che si sia nuovi
negli strumenti di chat AI o utenti esperti, queste guide aiutano a ottenere il
massimo dall'esperienza di conversazione.

Iniziare con la guida Per iniziare per imparare il layout dell'interfaccia e le
interazioni di base, quindi esplorare le Funzionalita per scoprire le query RAG,
il caricamento file e altre capacita che arricchiscono le conversazioni.

- **[Per iniziare](getting-started.md)** -- Iniziare la prima conversazione e
  imparare l'interfaccia di chat
- **[Funzionalita](features.md)** -- Esplorare RAG, caricamento file,
  integrazione API e altre capacita di chat
