# Chat

The chat interface is where you interact with AirGap AI Chatbot on a daily
basis. This section covers everything from sending your first message to using
advanced features like file attachments and context management. Whether you are
new to AI chat tools or an experienced user, these guides help you get the most
out of the conversation experience.

Start with the Getting Started guide to learn the interface layout and basic
interactions, then explore Features to discover RAG queries, file uploads, and
other capabilities that enhance your conversations.

- **[Getting Started](getting-started.md)** -- Start your first conversation
  and learn the chat interface
- **[Features](features.md)** -- Explore RAG, file upload, API integration,
  and other chat capabilities
